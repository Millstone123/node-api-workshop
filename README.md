# Node.js API Workshop

Build and test REST APIs with Node.js — no frameworks, no build step.

## Quick Start

```bash
git clone https://github.com/Millstone123/node-api-workshop.git
cd node-api-workshop
npm install
npm run server
```

## Commands

| Command | Description |
|---|---|
| `npm install` | Install dependencies and run first-time setup |
| `npm run server` | Start the mock API server |
| `npm run health` | Verify your environment |
| `npm test` | Run the test suite |

## Requirements

- Node.js 18+
- npm
